## 📊 Schema Details

### Table 1: gold_responses

**Columns:**
```sql
id              uuid PRIMARY KEY
title           varchar(255) NOT NULL
category        gold_category ENUM NOT NULL
body_template   text NOT NULL
tone_notes      text
is_active       boolean DEFAULT true
use_count       integer DEFAULT 0
avg_word_count  integer          -- From backfill metrics
avg_paragraph_count integer     -- From backfill metrics
sample_count    integer          -- Number of emails analyzed
created_at      timestamp DEFAULT now
updated_at      timestamp DEFAULT now
```

**Enums:**
```sql
gold_category = [
  'damaged_missing_faulty',
  'shipping_delivery_order_issue',
  'product_usage_guidance',
  'pre_purchase_question',
  'return_refund_exchange',
  'stock_availability',
  'partnership_wholesale_press',
  'brand_feedback_general'
]
```

---

### Table 2: knowledge_snippets

**Columns:**
```sql
id          uuid PRIMARY KEY
title       varchar(255) NOT NULL
type        snippet_type ENUM NOT NULL
category    snippet_category ENUM NOT NULL
content     text NOT NULL
tags        jsonb           -- Array of strings
source      varchar(255) DEFAULT 'backfill_pipeline'
is_active   boolean DEFAULT true
created_at  timestamp DEFAULT now
updated_at  timestamp DEFAULT now
```

**Enums:**
```sql
snippet_category = [same 8 categories as above]

snippet_type = [
  'policy',    -- Tone rules, terminology guidelines
  'fact',      -- Response patterns, statistics
  'guidance'   -- What to avoid, best practices
]
```
